# Preserved legacy assets

These first-party source assets were retained when the Flutter application was removed:

- `FlutterAssets/icons/tickle.png`: original master application icon.
- `AndroidResources`: launcher variants, colors, and widget resource definitions from the former Android target.

The active iOS app and widget catalogs live in `Tickle/Assets.xcassets` and `TickleWidget/Assets.xcassets`. Files in this directory are archival sources and are not copied into the application bundle.
